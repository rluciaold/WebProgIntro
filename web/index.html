Lesson 01: Using Wiggle

Sometimes you need to lose control. Using the Wiggle expression in AfterEffects will grant you the power to add and tweak randomness to any editable option, anywhere. In the following examples, I used a very simple setup. A grey wood texture as a 3D layer for the background, a camera to look at it, and a spot light. Wiggle can be used to add randomness to anything that allows an expression to be written, such as position, rotation, scale, brightness, hue, etc. In the following examples, I used wiggle to add randomness to the light's "Point of Interest", which is the point in which the light is aiming.
